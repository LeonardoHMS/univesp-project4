# Análise de acidentes - Projeto integrador UNIVESP IV

Código utilizado para visualização dos detalhes referente as inforamações de acidentes de trânsito fornecidos pela PRF.
https://www.gov.br/prf/pt-br/acesso-a-informacao/dados-abertos/dados-abertos-da-prf
